# Goit HomeWork | 09 JavaScript Template

## NPM Aşamaları

1. Projeyi klonlayın:

```bash
git clone <repository-url>
```

2. Proje dizinine gidin:

```bash
cd goit-js-hw-09-start
```

3. Gerekli paketleri yükleyin:

```bash
npm install
```

4. Projeyi başlatın:

```bash
npm run dev
```

5. Kodu paketleyin:

```bash
npm run build
```

6. Projeyi canlıya alın:

```bash
npm run deploy
```
